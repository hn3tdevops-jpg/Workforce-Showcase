# docs/planning

This folder contains the Workforce planning and execution framework.

## Folder map

- `00_README_FIRST.md` — entry point for the pack
- `reference/` — human-readable guide and PDF reference
- `templates/` — reusable templates and standards
- `workforce_studio/` — completed example set for Workforce Studio
- `copilot_install/` — install instructions and install prompt for Copilot

## Operational pattern

Use this framework in the following order:

1. Create a module spec.
2. Convert it into a master plan section.
3. Split the plan into atomic execution tasks.
4. Execute one checkbox per patch.
5. Record progress and status codes.
6. Use decision rules to avoid loops or scope creep.
7. Reuse the Workforce Studio files as a worked example.
