- `HN3T_MASTER_PLAN.md`

1. install the planning framework into the current repo
2. give Copilot a safer, more disciplined atomic execution workflow
3. standardize progress reporting and status vocabulary
4. add Workforce Studio planning artifacts without disrupting the existing repo
