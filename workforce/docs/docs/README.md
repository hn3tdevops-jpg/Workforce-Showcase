# Workforce Docs

This folder contains the canonical Workforce architecture and module specification documents that should be treated as primary design references.

## Canonical documents
- `AI_GATEWAY_MODULE_SPEC.md` — shared AI orchestration/provider boundary for Workforce.
- `COMMUNICATION_MODULE_SPEC.md` — shared thread/message/participant communication layer.
- `STUDIO_AI_INTEGRATION_SPEC.md` — how Studio uses Communication plus the AI Gateway.
- `WORKFORCE_STUDIO_IMPLEMENTATION_SPEC.md` — Studio module implementation spec.

## Suggested use
Use these files as the primary source when:
- defining module boundaries
- planning APIs and database tables
- building master-plan sections
- drafting Copilot handoff instructions
- reviewing architecture consistency

## Supporting/admin materials
Non-canonical support files, prompts, install guides, templates, plan inserts, and reference PDFs are placed in `docs/ADMIN/`.
