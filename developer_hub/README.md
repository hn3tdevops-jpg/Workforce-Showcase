# Workforce Developer Hub

This package contains a self-contained static developer portal page for the Workforce project.

## Files

- `index.html` — upload-ready static page

## What it includes

- Docs Library section
- Progress Reports section
- Milestones and Recent Activity panel
- Client-side search and filter controls
- Inline sample data you can edit directly inside the HTML file

## Quick setup

1. Upload `index.html` to your web host.
2. Rename it or route it however you want for your web tab.
3. Edit the arrays near the bottom of the file to replace placeholder content:
   - `docsData`
   - `reportData`
   - `milestoneData`
   - `activityData`

## Notes

This page is intentionally framework-free so it can be dropped into almost any existing static hosting setup.
